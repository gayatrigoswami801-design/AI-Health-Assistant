# AI-Health-Assistant
🏥 AI Health Assistant – Intelligent Healthcare Automation System

📌 Project Overview

AI Health Assistant is an advanced Python-based healthcare automation and patient management system designed to simplify healthcare operations through Artificial Intelligence, database management, and healthcare  architecture.

The system helps manage patient records, analyze symptoms, predict diseases, generate healthcare reports, maintain patient history, provide medicine reminders, and assist users during emergency situations.

This project demonstrates the integration of:

- Artificial Intelligence
- Healthcare Management
- Database Systems
- Automation
- Software Engineering
- Modular ERP Architecture

The application is designed as a mini Healthcare  platform capable of solving real-world healthcare management problems digitally and efficiently.

The main objectives of this project are:

# ▶️ How to Run the Project

1️⃣ Clone the repository using https://github.com/gayatrigoswami801-design/AI-Health-Assistant → 2️⃣ Open the project folder in VS Code → 3️⃣ Install required libraries using `pip install -r requirements.txt` → 4️⃣ Run the project using `python main.py` → 5️⃣ The AI Health Assistant menu will appear in the terminal.
